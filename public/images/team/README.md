# Team Photos

Drop team member photos in this folder, then reference them from the matching
member file in `src/content/team/<name>.md` by uncommenting and setting:

    photo: /images/team/<filename>.jpg

Until a `photo:` path is set, the member card shows an initials avatar.

Guidance:
- Use lowercase, hyphenated filenames matching the member (e.g. `jane-doe.jpg`).
- Square images work best (they are displayed in a circle); ~600×600px is plenty.
